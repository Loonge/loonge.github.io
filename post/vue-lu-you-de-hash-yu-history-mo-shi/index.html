Vue路由的Hash与History模式

2021-05-29 1 min read

Hash路由又叫前端路由，URL中会带有#号，且#号后面的参数不会被发送到后端。

History路由刷新会出现404，需要后端对服务器做一些配置。

Nginx服务器配置如下：

location / {
    try_files $uri $uri/ /index.html
}
